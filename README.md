#🛠️ Kullanılan Teknolojiler

- HTML / CSS
- ASP.NET Core 
- Visual Studio 

#📂 Proje Yapısı

- `uygulama/` klasörü: Projenin ana dosyaları
- `.sln` dosyası: Visual Studio çözüm dosyası
- `.vs/`: Geliştirici ortamına özel ayarlar (isteğe bağlı olarak gitignore'a alınabilir)
